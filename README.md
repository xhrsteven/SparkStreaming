# SparkStreaming


### CentOS 6.9 + 
### 需要大家在Linux的Hadoop用户跟目录下创建如下目录
### app 存放我们所有软件的安装目录

### data 存放我们的测试数据

### lib 存放我们开发jar包

### software 存放软件安装包的目录

### source 存放我们的框架源码

### root 用户密码 所有操作均有Hadoop用户完成
### 需要用root权限设置环境变量或相关操作 使用sudo command 

### Hadoop生态系统： cdh5.7.0
### 所有的软件生态下载 cdh5.7.0
### jdk 1.8.0
### spark 2.2
### scala: 2.11.8
